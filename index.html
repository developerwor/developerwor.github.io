3D

    1-1 프로젝트 개요

  • 정보

    • - 개인 프로젝트
    - 직업전문학교에서 진행한 메타버스 게임 프로젝트
    - 차량 전시관 공간 구현 비대면 시대에 공간과 시간의 제약을 받지 않음
    - 개인이 원하는 차량 정보를 습득 전시되어 있는 차량의 기능을 체감해보기 위한 환경 구현

  • 개발 환경

    • - 유니티(C#), 3DS Max, 포토샵

  • 작업기간

    • - 약 1개월(2022.10.11. ~ 2022,11.05)
    『1주차 : 컨셉 및 세계관 설정 / 배경 모델링(전시관, 운전면허장) / 소품 모델링
       2주차 : 캐릭터 모델링 / 차량 모델링 / 로비 씬 제작
       3주차 : 게임 UI 제작 / 차량 기능 구현
       4주차 : 차량 전시관 기능 구현 / 운전면허장 기능 구현』

  • 맡은 역할

    • - 기획, 프로그래밍, 3D 모델링 등의 모든 부분을 진행

    1-2 상세내용

  • 자동차의 이벤트

  • 전시관의 상호작용(영상정보 제공)

  • 공간 모델링

  • 소품 모델링

  • 최종결과

  • 화면

  • 프로젝트 성과

    • 개발자에게는 필요한 자질은 여러 가지가 있지만 제가 이번에 차량 메타버스 프로젝트를 진행함으로써 느낀 점은 개발하려고 하는 장르에 대한 지식이 매우 중요하다는 점입니다. Unity에서는 휠 콜라이더라는 자동차의 휠 기능을 사용할 수가 있었는데 휠 콜라이더로 자동차의 움직임을 자유자재로 제어하려고 하니 자동차의 용어나 구조에 대해 잘 몰라 제어를 제대로 할수가 없어서 난감했었습니다. 지식의 중요성뿐만 아니라 스크립트를 작성하는 데 있어서 여러가지 방법을 연구하고 다양한 소스를 보는 게 중요하다는 것을 몸소 체험하였습니다

VR

    1-1 프로젝트 개요

  • 정보

    • - 팀 프로젝트
    - 직업전문학교에서 진행한 메타버스 게임 프로젝트
    - 영화, '천공의 섬 라퓨타'에 영감을 얻어 기획한 프로젝트
    - 환상적인 공간을 구현하여 VR의 체험 효과를 극대화하는 것이 목표

  • 개발 환경

    • - 유니티(C#), VR(오큘러스, XRcore), 3Ds Max

  • 작업기간

    • - 약 2개월(2022.11.06. ~ 2022,12.31)
    『1주차 : 컨셉 및 세계관 설정 / NPC 모델링 / 배경 모델링
       2주차 : 소품 모델링 / 로비 씬 제작 / 게임 UI 제작
       3주차 : 배경 사운드 배치 / VR 환경 구축 / 오쿨루전 컬링을 이용한 최적화
       4주차 : 게임 내 모든 NPC 이벤트 구현 / NPC의 움직임과 행동 구현』

  • 맡은 역할

    • - 프로그래밍을 전담, 일부 캐릭터의 3D모델링 참여

    1-2 상세내용

  • NPC 이벤트 구현

    • - 게임 내의 6종의 이벤트 구현
    - 소지 아이템, 이벤트 진행 상태를 비교하여 이벤트 진행 구현

  • 개발 예시(성문 이벤트)

    • - 성문에 연료를 5개 투입 시, 성문이 열림

    - 성문이 열리는 연출 부분(코루틴 활용)

    - 결과 화면

  • NPC 동작 및 인공지능 구현

    • - 이동 인공지능(네비게이션)

    행동목적에 따라 네비게이션 메쉬의 목적지를 설정하여 진행상황에 맞는 NPC행동 구현

    상황에 맞는 NPC 동작 구현

  • 최적화

    • - 오클루전 컬링을 이용해 최적화 진행

    Play시 카메라에서 볼때 오브젝트 뒤에 가려진 부분은 계산에서 제외함으로 인해서 부담을 덜어준다

  • 3D 모델링 제작

  • 최종결과

    • 최종적으로 탈출에 필요한 퀘스트 수행 및 연료, 조타륜 획득 후 가족이 있는 섬으로 탈출

  • 화면

    • - 초기화면

    - 플레이 화면

  • 프로젝트 성과

    • 큰 공간에서의 상호작용과 퀘스트를 구현해봄으로서 기본적인 응용법, 안정적인 코딩능력을 향상 시킬수있었습니다. 또한 VR 환경을 구현함으로서 VR에 대한 이해력을 높일수가 있었습니다. 모델링을 제작할 때는 맵과 맵의 부수적인 소품들 그리고 캐릭터 간의 비율을 맞추는 것이 매우 중요하다는 것을 몸소 알 수 있었고 코드뿐만 아니라 모델링의 최적화도 게임을 플레이하는 데 있어 중요한 역할을 할 수 있다는 것을 깨달았습니다
